问题

问题描述

运行之后显示

浏览器没有自动打开

环境

  • macOS 10.12.5
  • python 3.4

原因

​ 这个问题由于最新的Mac OSX 10.12.5升级。

解决方案

  1. 生成配置文件

  2. 在终端中输入ipython3打开Python，生成密码的sha1密钥

    在文本框中输入并确认一次密码后记录sha1秘钥值，如’sha1：XXXXXXX’

    退出python

  3. vim打开配置文件jupyter_notebook_config.py，在最后加入

  4. 重启jupyter，重新打开网页即可
